## Portfolio web site
Source code for my portfolio web site:
[Demo](http://teh5.ru/new)

***
### Install

* `npm i`

### Run

* `npm start`

### Build

* `npm run build`

### Test

* `npm run test`

### Create JSDocs (in public/docs)

* `npm run doc`
